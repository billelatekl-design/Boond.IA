Cette app tourne sur http://localhost:3000 via un serveur Node.js qui fait le pont avec les APIs Anthropic et BoondManager.
🔑 Clé API Anthropic
Entrez votre clé Anthropic pour activer l'intelligence artificielle. Elle est stockée uniquement en mémoire et jamais sauvegardée sur disque.
Où trouver votre clé :
1. Allez sur console.anthropic.com/settings/keys
2. Cliquez Create Key
3. Copiez la clé (commence par sk-ant-...)
4. Collez-la ici et cliquez Sauvegarder
